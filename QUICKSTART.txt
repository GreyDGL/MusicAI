MusicAI Quick Start Guide
=========================

INSTALLATION
------------

Prerequisites:
- Python 3.8 or higher
- Git
- 4GB+ RAM (8GB+ recommended)

Quick Install:
1. Clone the repository:
   git clone https://github.com/GreyDGL/MusicAI.git
   cd MusicAI

2. Run the installation script:
   ./install.sh

3. Start the application:
   python app.py

4. Open your browser and go to: http://localhost:8080


BASIC FEATURES
--------------

1. Generate Music
   - Go to "Generate" in the navigation menu
   - Enter a prompt (e.g., "A peaceful piano melody")
   - Select model size (Small recommended for first use)
   - Click "Generate Music"

2. Evaluate Music
   - Go to "Evaluate" tab
   - Select a generated music file
   - Rate various aspects (1-5 stars)
   - Add comments
   - Submit evaluation

3. Sync with Collaborators
   # Get latest updates
   git pull origin main

   # Save your evaluations
   git add .
   git commit -m "My evaluations"
   git push origin main


EXAMPLE PROMPTS
---------------
- "Upbeat jazz piano solo"
- "Epic orchestral battle music"
- "Calm ambient nature sounds"
- "90s rock with electric guitar"


AVAILABLE MODELS
----------------
- Small (300MB): Fast, good quality
- Medium (1.5GB): Balanced performance
- Large (3.3GB): Best quality


TROUBLESHOOTING
---------------
- Application won't start: Try python3 app.py
- Port in use: Change port in config.py
- Memory issues: Use the Small model


IMPORTANT FILES
---------------
- music/ - Generated music files
- database.db - Evaluation data
- config.py - Settings
