MusicAI Quick Start Guide
=========================

PREREQUISITES
-------------
Before starting, make sure you have:
- Python 3.8 or higher installed
- Git installed
- 4GB+ RAM (8GB+ recommended)
- GitHub account (create free at github.com)

INSTALLATION
------------

Option A: If you received this as a ZIP file:
----------------------------------------------
1. Unzip the folder to your Desktop or Documents folder

2. Open Terminal (Mac) or Command Prompt (Windows):
   - Mac: Press Cmd+Space, type "Terminal", press Enter
   - Windows: Press Windows key, type "cmd", press Enter

3. Navigate to the unzipped folder:
   Type: cd
   Then drag the MusicAI folder onto the Terminal window
   Press Enter

4. Connect this folder to GitHub (ONE-TIME SETUP):
   git init
   git remote add origin https://github.com/GreyDGL/MusicAI.git
   git branch -M main

   # Set up your identity (use your actual name and email):
   git config --global user.name "Your Name"
   git config --global user.email "your-email@example.com"

5. Run the installation:
   ./install.sh  (Mac/Linux)
   OR
   python install.py  (Windows)

6. Start the application:
   python app.py
   (If that doesn't work, try: python3 app.py)

7. Open your browser and go to: http://localhost:8080

Option B: If cloning from GitHub directly:
-------------------------------------------
1. Open Terminal/Command Prompt
2. Clone the repository:
   git clone https://github.com/GreyDGL/MusicAI.git
   cd MusicAI
3. Continue from step 5 above

DAILY USE
---------

Starting Your Work Session:
1. Open Terminal/Command Prompt
2. Navigate to MusicAI folder (cd then drag folder)
3. Get latest changes from others:
   git pull origin main
4. Start the app:
   python app.py
5. Open browser to: http://localhost:8080

Ending Your Work Session:
1. Close the browser tab
2. In Terminal, press Ctrl+C to stop the app
3. Save and share your work:
   git add .
   git commit -m "Added my evaluations"
   git push origin main

BASIC FEATURES
--------------

1. Generate Music:
   - Click "Generate" in menu
   - Type what kind of music you want (see examples below)
   - Choose "Small" for model size (fastest)
   - Click "Generate Music"
   - Wait 10-30 seconds

2. Evaluate Music:
   - Click "Evaluate" in menu
   - Select a music file
   - Listen to it
   - Rate each aspect with stars (1-5)
   - Add comments if you want
   - Click "Submit Evaluation"

3. View Dashboard:
   - Click "Dashboard" to see statistics
   - See all evaluations
   - Export data for analysis

COMMON PROBLEMS & SOLUTIONS
---------------------------

"Command not found" errors:
- Make sure Python is installed: python --version
- Make sure Git is installed: git --version
- Try python3 instead of python

"Permission denied" pushing to GitHub:
- Make sure you're added as a collaborator
- Check your GitHub username/password

Application won't start:
- Try: python3 app.py
- Make sure you're in the MusicAI folder
- Check that installation completed

Can't see the website:
- Make sure the app is running (green text in Terminal)
- Try refreshing: http://localhost:8080
- Check no other program is using port 8080
